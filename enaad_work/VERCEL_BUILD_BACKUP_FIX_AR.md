# إصلاح فشل نشر Vercel بعد إضافة النسخ الاحتياطي

- أزيل منع النشر عند غياب `EMAIL_FROM`.
- عند عدم ضبط `EMAIL_FROM` يستخدم زر النسخ الاحتياطي مرسل Resend التجريبي `onboarding@resend.dev`.
- هذا المرسل التجريبي يعمل فقط عندما يكون البريد المستلم هو نفس البريد المرتبط بحساب Resend.
- عند إضافة نطاق موثّق لاحقًا، يستخدم النظام قيمة `EMAIL_FROM` تلقائيًا.
- تبقى أخطاء الإعداد واضحة داخل زر النسخ الاحتياطي بدل تعطيل نشر الموقع بالكامل.
