# تقرير النسخة النهائية — تفعيل الإدارة والإصلاحات الحرجة

## الحساب الإداري

- الحساب الوحيد: `enaadx@gmail.com`.
- إذا كان الحساب موجودًا وقت النشر، تقوم عملية seed بضبط `role=admin` و`emailVerified=true`.
- إذا أُنشئ الحساب بعد النشر، تتم المزامنة تلقائيًا داخل الخادم عند أول تسجيل دخول موثّق.
- يتم إنشاء/تفعيل `adminAllowlist` وتسجيل العملية في سجل التدقيق.
- لا يقبل الخادم دورًا إداريًا قادمًا من المتصفح، ولا يرقّي أي بريد آخر.
- بعد الدخول يتم توجيه الحساب الإداري إلى `/admin` وتظهر وصلة «لوحة التحكم» في القوائم.

## لوحة التحكم

الأقسام: الرئيسية، الإحصائيات، المستخدمون، الأقسام، المحتوى التعليمي، التحديات والمنافسات، المستويات والترقيات، الشارات والإنجازات، محتوى الواجهة، الوسائط، سجل العمليات، النسخ الاحتياطي، والإعدادات.

## قاعدة البيانات والرياضيات

- الترحيلات تعمل قبل seed والبناء.
- إصلاح جدول `rateLimit` مضمّن.
- بنك الرياضيات يحتوي على 30 سؤالًا للأساسيات و30 للكسور و30 للجبر.
- نقص الأسئلة يعرض رسالة عربية بدل انهيار Server Component.

## فحص النشر

أُضيف `release:verify` إلى خط بناء Vercel. يفشل النشر بدل إظهار Ready إذا كانت الجداول الحرجة أو allowlist أو بنوك الرياضيات ناقصة.
